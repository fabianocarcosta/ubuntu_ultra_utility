🚀 Como Instalar no Ubuntu

1. Via Pacote .DEB (Recomendado)

O formato .DEB integra o aplicativo diretamente ao seu sistema e cria um atalho no menu.

# 1. Baixe o arquivo .deb na seção de Downloads acima
# 2. Abra o terminal na pasta do download e execute:
sudo apt update
sudo apt install ./ubuntu_ultra_utility_1.0.0_amd64.deb

2. Via AppImage (Portátil)

Não precisa de instalação. Basta baixar e rodar!

  • Baixe o arquivo .AppImage.
  • Clique com o botão direito no arquivo e vá em Propriedades.
  • Na aba Permissões, marque a caixa "Permitir execução do arquivo como programa".
  • Dê um clique duplo para abrir o aplicativo.
